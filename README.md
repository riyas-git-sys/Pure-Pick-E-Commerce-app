# 🛍️ PurePick - Modern E-Commerce Platform

![PurePick Demo](https://riyaspurepick.ccbp.tech/) <!-- Add your demo gif here -->

## ✨ Features

### 🛒 Core Functionalities
- **Product Catalog**
  - Category-wise product listings
  - Detailed product pages with images, specs, and ratings
  - Similar product recommendations
- **Shopping Cart**
  - Add/remove items
  - Quantity adjustment
  - Cart summary with total calculation
- **Checkout System**
  - Multi-step checkout process
  - Address and payment information collection
  - Order confirmation

### 🎛️ User Features
- **Authentication**
  - JWT-based login/logout
  - Protected routes
- **Order Management**
  - Order history
  - Order status tracking
- **Interactive UI**
  - Product filtering
  - Smooth animations
  - Responsive design
